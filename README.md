# CGPA Calculator (C++)

## Description
This is a simple C++ program that calculates the CGPA of a student based on the credit hours and grade points of each course.

## Features
- Takes the number of courses as input.
- Accepts credit hours and grade points for each course.
- Calculates total credits.
- Calculates total grade points.
- Computes and displays the final CGPA.

## Requirements
- C++ Compiler (G++, Code::Blocks, Dev-C++, Visual Studio)

## How to Run
1. Compile the program:
   ```
   g++ cgpa.cpp -o cgpa
   ```
2. Run the program:
   ```
   ./cgpa
   ```

## Sample Input
```
Enter number of courses: 3

Course 1
Credit Hours: 4
Grade Point: 10

Course 2
Credit Hours: 3
Grade Point: 9

Course 3
Credit Hours: 3
Grade Point: 8
```

## Sample Output
```
Total Credits = 10
Total Grade Points = 91
CGPA = 9.1
```

## Formula Used
```
CGPA = Total Grade Points / Total Credits
```
